# How I Debugged No-Display Boot Problems

This note tracks the steps I use when a machine has no signal on boot.

1. Verify PSU and front-panel power/ reset connections.
2. Reseat RAM, GPU, and display cable.
3. Confirm BIOS/UEFI output target and monitor refresh compatibility.
4. Clear CMOS and test with minimum hardware set.
5. Update BIOS only after confirming stable POST conditions.
